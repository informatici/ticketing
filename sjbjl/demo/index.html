about

sjbjl, as its name says, is a jQuery Based Library, putting together some stuff to improve your web applications user experience in an easy way.
It's based on jQuery, it has all jQuery UI plugins embedded in minified version.
It contains also some other selected plugins to do few good things with your html in little amount of time.
I decided to re-use all my code and started this library in order to make unpainful the upgrading and editing of my web applications.

the author

sjbjl is created by rodenic
